1. **Regular Expressions Regex**
  - [A-Z] : all capital alphabet included
  - [a-z] : all small letters included
  - [abc] : only a, b, c included
  - [a-d] : a,b,c,d letters included
  - [0-9] : 0 to 9 all digit included
  - [^a-z] : a to z excluded
  - [^char] : character excluded
